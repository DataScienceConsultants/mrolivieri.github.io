# Mr. Olivieri Urban Artist Page

GitHub Pages-ready files for an urban-style artist page.

## Files
- `mr-olivieri.html`
- `mr-olivieri.css`
- `mr-olivieri.js`

## Image files to add
Create an `images` folder and upload:

- `mr-olivieri-hero.jpg`
- `artwork-1.jpg`
- `artwork-2.jpg`
- `artwork-3.jpg`
- `mural-background.jpg`

## How to use
Upload these files to your GitHub repository.  
Open the page as:

`https://yourusername.github.io/yourrepo/mr-olivieri.html`

## Update before publishing
Replace:

`YOUR_EMAIL_HERE`

with the real booking/contact email.
